This software is aimed to read satellite altimetry data of the netCDF format.
The netCDF4 lib is used under python3.
This software is aimed to:
1: Read Sentinel-3A SRAL data, including the waveform data, analysis data and show data in
figures.
2: Apply AI algorithm to the waveform data to classify sea ice and sea water in the arctic ocean.
3: ...
